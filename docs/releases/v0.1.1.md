# v0.1.1 — 单一主程序 / Single executable

## 中文

此版本把一键启动器和主题切换器统一为 Release 中唯一的 `Codex自定义主题.exe`。直接运行打开主题切换器，使用 `--launch` 进入一键启动模式；桌面和开始菜单快捷方式都指向同一个文件。

发布校验会拒绝旧的 `CodexThemeSwitcher.exe` 和 `Codex自定义主题启动器.exe`，避免重复分发同一二进制。程序仍为 Windows x64 自包含单文件，无需单独安装 .NET Desktop Runtime。

## English

This release unifies one-click launch mode and the theme switcher in the sole Release executable, `Codex自定义主题.exe`. Running it directly opens the switcher, while `--launch` starts or activates themed Codex. Desktop and Start Menu shortcuts both target this one file.

Release verification now rejects the obsolete `CodexThemeSwitcher.exe` and `Codex自定义主题启动器.exe` duplicates. The application remains a self-contained Windows x64 single-file executable.
