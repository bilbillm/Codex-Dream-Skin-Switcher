# v0.1.0 — 首个公开版本 / First public release

## 中文

这是 Codex Dream Skin Switcher 的首个公开版本，把 Dream Skin `3.1.4-angelina` 运行时、两套 Angelina 明暗主题、主题预览/导入/热切换 GUI 和 Codex 一键启动模式打包为可直接解压使用的 Windows x64 工具。

### 主要内容

- 自包含 `CodexThemeSwitcher.exe`，无需单独安装 .NET Desktop Runtime。
- 无参数打开主题切换器，`--launch` 打开一键启动模式。
- 健康 CDP 会话快速唤醒；watcher 缺失时自动恢复完整运行链路。
- Angelina Gravity Field 白天主题。
- Angelina Midnight Gravity 黑夜主题。
- 左/右/底部面板、聊天气泡、应用内二级菜单和右上角置顶摘要卡片磨砂玻璃。
- “回答问题”多题选择卡与输入框使用同系磨砂玻璃，并统一为白色高对比文字与图标。
- 桌面和开始菜单快捷方式安装/卸载脚本。
- 中文主 README 与完整英文 README。
- 发布 ZIP 清单、SHA-256 和敏感凭据模式验证。

## English

The first public release bundles the Dream Skin `3.1.4-angelina` runtime, two Angelina light/dark themes, the preview/import/hot-switch GUI, and one-click Codex launch mode as a self-contained Windows x64 package.

Highlights include healthy-session fast activation, watcher recovery, frosted side/right/bottom panels, chat bubbles, in-app secondary menus, the top-right pinned-summary card, a composer-matched white-text multi-question response card, shortcut installers, bilingual documentation, and verified SHA-256 release artifacts.

This is not an official Codex theme mechanism. Native Electron application menus are not styled, future Codex DOM changes may require compatibility updates, and bundled fan-art assets require independent rights review before commercial use or redistribution.
