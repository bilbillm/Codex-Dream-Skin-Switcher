# v0.2.0 — 统一启动控制台 / Unified launch console

## 中文

`Codex自定义主题.exe` 现在只打开一个统一控制台：选择主题后可单独应用，或点击“启动 Codex”让它先保存所选主题再启动或唤醒 Codex。主题选择本身不再自动改变正在运行的 Codex。

未连接的情况下，“应用主题”会安全保存主题以供下次启动使用；已连接时会立即热切换。启动成功后，控制台默认最小化到系统托盘；设置中可改为保持窗口打开。

安装器现在只创建一个无参数的“Codex 自定义主题”快捷方式，并清理旧的“Codex 主题切换器”开始菜单入口。

## English

`Codex自定义主题.exe` now opens one unified console. Select a theme, apply it independently, or click “Launch Codex” to save the selection before starting or activating Codex. Selecting a theme alone no longer changes a running Codex session.

When disconnected, Apply safely saves the theme for the next launch; when connected, it hot-switches immediately. After a successful launch, the console minimizes to the system tray by default, with a settings option to keep it open.

The installer now creates one argument-free “Codex 自定义主题” shortcut and removes the legacy “Codex 主题切换器” Start Menu entry.
